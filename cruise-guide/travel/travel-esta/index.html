미국 ESTA 신청 방법 완전 가이드

미국 여행·경유 전 ESTA 신청은 필수입니다. 온라인으로 간단히 신청할 수 있지만, 공식 사이트가 아닌 사기 대행 사이트가 많습니다. 정확한 방법을 알려드립니다.

📋 ESTA란?

ESTA(Electronic System for Travel Authorization)는 미국 비자 면제 프로그램(VWP) 참여국 국민이 미국 방문 전 온라인으로 받아야 하는 여행 허가입니다. 비자가 아니며, ESTA 승인이 곧 입국 보장은 아닙니다.

  • 유효기간: 2년 또는 여권 만료일 중 빠른 날
  • 허용 체류: 1회 최대 90일
  • 비용: $21 (사기 사이트는 $100 이상 청구)

📝 신청 방법 (단계별)

  1. 공식 사이트 접속: esta.cbp.dhs.gov (반드시 이 주소만)
  2. New Application 클릭
  3. 여권 정보 입력 (여권 스캔 자동 입력 가능)
  4. 여행 정보 입력 (미국 내 주소 등)
  5. 보안 질문 답변 (전염병 보유 여부 등)
  6. $21 결제 (신용카드)
  7. 승인 확인 (보통 즉시~72시간)
  • ⚠️ 구글에서 "ESTA 신청" 검색 시 광고로 올라오는 대행 사이트 주의 — 공식은 $21
  • ✅ 주소창에 직접 esta.cbp.dhs.gov 입력

✅ 승인 상태

  • Authorization Approved: 여행 가능
  • Authorization Pending: 72시간 내 재확인
  • Travel Not Authorized: 거절 → 미국 대사관 비자 신청 필요

🔄 갱신 및 주의사항

  • 여권 갱신 시 ESTA 재신청 필요
  • 이름·여권번호 변경 시 재신청
  • 출발 최소 72시간 전 신청 권장
  • 캐나다·카리브해 크루즈 미국 경유 시도 ESTA 필요
