🔌 해외여행 전기 어댑터·전압 완전 가이드

나라별 플러그 타입, 전압, 멀티어댑터 선택 가이드

해외에서 콘센트가 맞지 않아 충전을 못하는 경우가 생각보다 많습니다. 어댑터를 미리 준비해야 하고, 전압도 확인해야 합니다. 한국은 220V이지만 미국은 110V — 잘못 사용하면 기기가 망가질 수 있습니다.

🔌 세계 주요 플러그 타입

타입사용 국가전압한국과 호환
Type A (평행 2핀)미국, 캐나다, 멕시코, 일본110~120V❌ (어댑터 필요)
Type C (둥근 2핀)유럽 대부분, 한국, 러시아220~240V✅ (대부분 호환)
Type G (사각 3핀)영국, 홍콩, 싱가포르220~240V❌ (어댑터 필요)
Type I (사선 2핀)호주, 뉴질랜드220~240V❌ (어댑터 필요)
Type B (3핀 납작)미국, 캐나다110V❌

⚡ 전압 & 변압기 필요 여부

⚠️ 미국·캐나다·일본 여행 시 전압 주의
이 나라들은 110V를 사용합니다. 한국 가전(220V용)을 그냥 꽂으면 고장납니다. 단, 스마트폰 충전기, 노트북 어댑터 대부분은 100-240V 자동 변환이라 어댑터만 있으면 됩니다. 헤어드라이어, 고데기는 반드시 사양 확인하세요.
💡 기기 사양 확인 방법
충전기나 가전 제품 본체 또는 어댑터에 "Input: 100-240V~" 라고 적혀 있으면 전 세계 어디서든 변압기 없이 사용 가능합니다. "220V only"라면 변압기가 필요합니다.

🛒 멀티어댑터 선택 가이드

  • 전 세계 호환 4-in-1 멀티어댑터 (쿠팡에서 1~2만원)
  • USB-A + USB-C 포트 내장 어댑터로 멀티 충전
  • GaN 충전기 + 멀티어댑터 조합이 가장 컴팩트
  • 접지 핀(3핀) 장비가 있다면 접지 지원 어댑터 필요
  • 호텔 욕실은 전동 면도기용 소켓만 제공하는 경우 多

📱 여행 필수 전자기기 충전 체크리스트

  • 스마트폰 충전기 (C타입 or 라이트닝)
  • 보조배터리 (10,000~20,000mAh, 100Wh 이하)
  • 노트북 어댑터 (여행 중 업무 필요 시)
  • 카메라 배터리 충전기 + 여분 배터리
  • 멀티탭 (멀티어댑터 1개 + 멀티탭으로 여러 기기 동시 충전)
