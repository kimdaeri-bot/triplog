항공편 지연·취소 시 대처법 완전 가이드

항공지연취소보상

갑자기 항공편이 지연되거나 취소되면 당황하기 마련입니다. 하지만 알고 있으면 보상을 받을 수 있습니다.

⚖️ 승객 권리

  • EU261 규정: 유럽 출발·도착 항공편 3시간+ 지연 시 €250~600 보상
  • 국내선: 3시간 이상 지연 시 운임의 10~30% 배상
  • 취소 시: 전액 환불 또는 대체편 제공 의무

🚀 지연 발생 시 즉시 할 것

  1. 카운터에서 지연 사유 확인서 발급 요청
  2. 항공사 앱·웹 대체편 즉시 탐색
  3. 식사 바우처 요청 (2시간+ 지연 시)
  4. 호텔 숙박 요청 (하룻밤 이상 지연 시)

🛡️ 여행자 보험 청구

  • 지연 증명서·영수증 전부 보관
  • 통상 3시간 이상부터 청구 가능
  • 식비·숙박비·교통비 실비 보상
